## Alceo Token

Definimos un token bajo el estándar ERC20 de Ethereum con las siguientes características:

	* Nombre: Alceo TOken
	* Símbolo: ALT
	* Cantidad total: 80.000.000
	* Décimales: 8

Además se establece una distribución inicial como sigue:

	* Equipo: 5%
	* Fundadores: 5%
	* Reservado: 10%
